# Overview
This page was created for Assignment 1 to help people understand Sustainable Development Goal 13: Climate Action.It shares information about the 2025 Climate Action Hackathon, an event dedicated to solving problems such as the urban heat island effect and flooding through innovative ideas and teamwork.The website is built with HTML5 and CSS3 and runs smoothly on all devices. It is easy to use, clear in its content, and user-friendly for everyone.

# Features
* Hero Section: Event name, date, and “Register Now” button.
* About Section: Overview of purpose, goals, and expected outcomes.
* SDG 13 Connection: Explanation of how proposed solutions address climate risks.
* Event Schedule: Key event dates and activities.
* Speakers & Mentors: Cards with names, roles, and placeholder images.
* Tracks/Challenges: Four main focus areas – Heat Mitigation, Flood Early Warning, Green Infrastructure, and Community Apps.
* Resources: Placeholder dataset and API links.
* Registration Form: Basic HTML form (no JavaScript) with name, email, role, and interest.
* Sponsors & Partners: Logo grid display.
* Contact & Socials: Email and social media links for inquiries.

# Accessibility & Design
* Semantic HTML for structure and screen reader support.
* High contrast color palette for readability.
* Skip-to-content link for keyboard navigation.
* Responsive layout using Flexbox and CSS Grid.
* Hover and focus states for user interaction.
